# excel-sales-analysis-dashboard
Sales data analysis using Excel PivotTables and dashboard visualization

# Overview
This project showcases my ability to analyze and visualize sales data using Microsoft Excel.
I used PivotTables and dashboards to extract meaningful business insights.

#Objectives
- Analyze sales performance by category
- Compare revenue vs quantity sold
- Identify top-performing product categories
- Present insights using a dashboard

 #Tools Used
- Microsoft Excel
- PivotTables
- Charts & Dashboard Design

#Project Structure
- **Raw_Data**: Original dataset
- **Analysis**: PivotTables used for summarizing data
- **Dashboard**: Visual representation of key insights

 #Key Insights
- Technology generates higher revenue despite lower quantity sold
- Furniture has higher sales volume but lower profitability
- Sales performance varies significantly across categories

#Features
- Clean and structured dataset
- Multiple PivotTables for analysis
- Interactive dashboard for visualization

 #Skills Demonstrated
- Data cleaning and organization
- Data analysis using PivotTables
- Data visualization
- Business insight generation
